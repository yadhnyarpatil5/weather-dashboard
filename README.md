# Weather Dashboard Application

## Project Description

The Weather Dashboard Application is a Python-based weather information system that fetches real-time weather data using the OpenWeatherMap API. The application allows users to check current weather conditions, view 5-day forecasts, and convert temperatures between Celsius and Fahrenheit.

This project demonstrates API integration, JSON data handling, HTTP requests, external library usage, and error handling.
